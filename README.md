## MSL Logger
Decrypts Netflix MSL messages and logs them \
**Currently only tested on netflix.com**

## Installation
+ Install [Tampermonkey](https://www.tampermonkey.net/)
+ Install this script from [Greasy Fork](https://greasyfork.org/en/scripts/560305-msl-logger)

Readable MSL requests and responses should now be displayed in the Developer Tools Console tab.
